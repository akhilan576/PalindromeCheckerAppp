# PalindromeCheckerAppp 🔤

A Java-based Palindrome Checker application that verifies whether a given string reads the same forwards and backwards.

## 📋 Project Overview

This is a beginner-level Java project built to practice string manipulation, conditional logic, and basic application structure. It takes a user input string and determines if it is a palindrome.

## 🚀 Features

- Accepts a string input from the user
- Checks if the string is a palindrome (case-insensitive)
- Displays a clear result message to the user
- Simple and clean implementation using Java

## 🛠️ Tech Stack

- **Language:** Java
- **IDE:** IntelliJ IDEA
- **Concepts Used:** String manipulation, loops/recursion, conditionals

## 🧑‍💻 How to Run

1. Clone the repository:
   git clone https://github.com/akhilan576/PalindromeCheckerAppp.git
2. Open in IntelliJ IDEA or any Java IDE.
3. Run the main Java file.
4. Enter a word or phrase to check if it is a palindrome.

## 💡 Example

Input: racecar
Output: "racecar" is a Palindrome!

Input: hello
Output: "hello" is NOT a Palindrome.
